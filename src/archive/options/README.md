## Archived Options Modules

This directory contains the options-specific system that was removed from the active app during the stocks-only pivot.

Archived areas:
- options calculator and strategy pages
- options quant dashboards and APIs
- websocket quant engine
- Black-Scholes and regime helper libraries
- Gemini prompts written for options analysis
- Python HMM prototype used by the older quant flow

These files are intentionally preserved for reference and possible selective reuse, but they are excluded from TypeScript compilation and are not part of the active product.
